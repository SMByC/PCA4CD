PCA4CD

The PCA4CD is a Qgis plugin for generate and analysis the principal components (PCA) for change detection

Documentation

Home page documentation: https://smbyc.bitbucket.io/qgisplugins/pca4cd

Source code

The official version control system repository of the plugin: https://github.com/SMByC/PCA4CD
